üìñ Template Help

You can customize the SMS alert format using placeholders:

  • {status} ‚Üí shows alert status (firing/resolved)
  • {summary} ‚Üí shows the alert summary
  • {description} ‚Üí optional description if available
  • {alertname} ‚Üí name of the alert rule
Examples:

Short SMS:
{status} {summary}
‚Üí resolved Instance1 CPU above 90%

Detailed SMS:
{status} - {alertname}: {summary} | Info: {description}
‚Üí firing - CPU_Alert: CPU above 90% | Info: Host overloaded

‚ö†Ô∏è Keep SMS short for better delivery (<160 chars). Long templates may split into multiple messages.
